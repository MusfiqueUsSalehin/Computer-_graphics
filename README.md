JumpQuest 🎮

A 3D platformer game developed as part of my Computer Graphics course project using Python and PyOpenGL.

🕹️ Main features:

☑️ Dynamic platform generation

☑️ Player movement with running and jumping mechanics

☑️ Obstacles like walls, enemies, and collectible coins

☑️ Collision detection and scoring system

☑️ Basic attack system with health and lives

☑️ Camera that follows the player in perspective view




>>Tech Stack:

Language: Python

Graphics Library: PyOpenGL (OpenGL for Python)



>>Learning Outcomes:

This project helped me explore the fundamentals of 3D rendering, transformations, and real-time interaction with OpenGL while building a playable game.
